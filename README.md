# Pane Paint

## Overview
Pane Paint is a JavaFX painting application that allows users to create and modify drawings in an intuitive, user-friendly interface. This application is part of the OBJ2000 course assignment.

## Technologies Used
- **Java**: This application is built using Java 19.
- **JavaFX**: The graphical user interface is developed using JavaFX version 19.0.2.
- **Maven**: The project utilizes Maven for dependency management and build automation.
- **JUnit**: For testing, JUnit version 5.9.2 is used.

## Main Class
The application entry point is `HelloApplication`, which is located in the `com.example.objoblig` package.

## Features
- Create, edit, and save drawings
- User-friendly interface with a variety of colors and brush options
- Supports drawing on a canvas pane
- Intuitive painting experience
